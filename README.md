# 🔐 Password Audit

## 📌 Description

A Python tool designed to evaluate password strength and simulate brute-force attack scenarios.

## 🎯 Objective

Demonstrate how weak passwords can be exploited and highlight the importance of strong authentication practices.

## ⚙️ How it works

* Analyzes password complexity (length, characters, patterns)
* Simulates brute-force attempts
* Provides feedback on password strength

## ▶️ How to run

```bash
python main.py
```

## 📊 Example Output

Password strength: Weak
Estimated brute-force time: 2 minutes

## 🧠 Security Context

Represents how attackers exploit weak credentials and how SOC teams can identify authentication risks.
